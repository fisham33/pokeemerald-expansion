# Quick Start Guide

Get your Pokemon Emerald Live Tracker running in 5 minutes!

## âš¡ Fast Setup (First Time)

### 1. Build Your ROM

```bash
cd /path/to/pokeemerald-expansion
make clean && make && make syms
```

### 2. Copy Symbol File

```bash
cp pokeemerald.sym mgba_scripts/
```

### 3. Generate Data Tables

```bash
cd mgba_scripts
./generate_data_tables.sh
```

### 4. Test the Frontend (Optional)

Before running the actual tracker, test that the frontend works:

```bash
cd frontend
# Copy sample data to test with
cp ../tracker_data_sample.json ../tracker_data.json
# Open index.html in your browser
```

You should see a sample team with Blaziken, Swampert, etc.

### 5. Load in mGBA

1. Open mGBA
2. Load ROM: `File > Load ROM` â†’ `pokeemerald.gba`
3. Load Script: `Tools > Scripting... > File > Load script` â†’ `mgba_scripts/tracker.lua`
4. Check console for: `Tracker initialized successfully!`

### 6. Open Frontend

Double-click `mgba_scripts/frontend/index.html`

### 7. Play!

Start or load your save. The tracker updates every second!
